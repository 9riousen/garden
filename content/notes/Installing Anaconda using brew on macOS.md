# 설치
```bash
brew install anaconda
```

# 소감
- <https://anaconda.com> 에 Mac용 installer (`*.pkg`) 도 제공된다(intel, Apple Silicon)
- 따라서 이게 좋은건지 확신이 없다
	- Alfred 에서 `navigator` 같은게 app으로 인식되면 좋겠는데 installer로 설치하면 되려나?
